# Roadmap

This is the public roadmap for DevContract. It focuses on the product work that is visible and useful to outside users.

## Now

- tighten relay operations and self-hosting guidance
- improve cross-platform test coverage
- keep install and release verification simple and trustworthy

## Next

- richer conflict previews and recovery UX
- stronger extension diagnostics and onboarding guidance
- more admin-grade relay tooling around membership and audit review

## Later

- deeper IDE support
- stronger release provenance and SBOM generation
- optional hosted operational surfaces after the self-hosted path is well proven

## Not Promised

This roadmap is directional. It is not a contract for dates, hosted billing, enterprise features, or roadmap guarantees.
